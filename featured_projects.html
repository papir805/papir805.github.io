Bot Battles Player Analysis

Bot Battles is a mobile game available in North America. This analysis examines player demographs and in-game activity over one year to investigate activity patterns and establish player profiles. An introduction to the game can be found here. The analysis is broken into three parts:
  • Part 1: Loading and cleaning the data in PostgreSQL.
  • Part 2: An exploratory data analysis to familiarize myself with the game and its data.
  • Part 3: Using hypothesis testing and regression modeling to answer the question about correlation.
Python and PostgreSQL
Data Cleaning, Visualization, Regression, Hypothesis Testing
SQLAlchemy, Pandas, Matplotlib, Seaborn, Statsmodels, SciPy

ElectroniCart Sales Analysis

ElectroniCart is an online electronics retailer who has collected a wealth of data from 100k+ orders since being established in 2019. This project aims to provide insights and recommendations on trends in sales, refunds, and performance of the recently introduced loyalty program.
