National Tutoring Prices Dashboard

Goal: To practice creating dashboards in Tableau.

Tableau
Visualization

Background

In another project, I built a tool that mined tutoring prices from Craigslist postings and stored them in a local PostgreSQL database. Using that data, I created an interactive dashboard which can be used to understand tutoring prices at the national, regional, state, and local level.

Note: The dashboards aren't optimized for tablets or mobile devices and is best viewed on a desktop or laptop.
