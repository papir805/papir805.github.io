National Tutoring Prices Dashboard

Goal: To create a dashboard of math tutoring prices in Tableau.

Tableau
Visualization

Background

In another project, I built a tool that mined tutoring prices from Craigslist postings and stored them in a local PostgreSQL database.  Using that data, I created an interactive dashboard that can be used to understand tutoring prices at the national, regional, state, and local levels.

Note: The dashboards aren't optimized for tablets or mobile devices and are best viewed on a desktop or laptop.
